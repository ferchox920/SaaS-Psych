# S8_8.3 - Visualizacion OpenAPI local con Swagger UI

## Paso
`S8_8.3`

## Objetivo
Agregar visualizacion local de la especificacion OpenAPI via Swagger UI, integrada con `docs/openapi.yaml`, sin alterar rutas existentes del API.

## Estado previo detectado
- La especificacion OpenAPI ya existia en `docs/openapi.yaml` (S8_8.2).
- No habia UI integrada para navegar la spec desde la API.
- README no documentaba una ruta local de docs visuales.

## Cambios realizados
- Se agrego handler de documentacion:
  - `GET /docs` -> pagina Swagger UI (CDN) apuntando a `/docs/openapi.yaml`
  - `GET /docs/openapi.yaml` -> sirve la spec desde `docs/openapi.yaml`
- Se extendio wiring en `internal/http/server.go` para registrar esas rutas.
- Se agregaron tests minimos de handlers:
  - `DocsUI` responde 200 con HTML
  - `OpenAPISpec` responde 200 con body de spec
- Se actualizo README con seccion `API Docs (Swagger UI)` y URLs locales.
- No se modificaron rutas `/api/v1/*` existentes.
