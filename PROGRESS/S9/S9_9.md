# S9_9 - Sincronizacion documental de roadmap y version Go

## Paso
`S9_9`

## Objetivo
Sincronizar la documentacion del roadmap (`SPRINTS.md`) con el estado real de implementacion de `PROGRESS_INDEX` y alinear la version de Go documentada en `README` con `apps/api/go.mod`.

## Estado previo detectado
- `PROGRESS/PROGRESS_INDEX.md` ya registraba avances hasta `S8_8.3` (OpenAPI + Swagger UI + tracing + observabilidad + CI).
- `SPRINTS.md` tenia contradicciones: `S7` figuraba `IN PROGRESS` y no incluia `S8`.
- `README.md` declaraba `Go 1.23+` mientras `apps/api/go.mod` define `go 1.24.0`.

## Cambios realizados
- Se actualizo `SPRINTS.md` para reflejar el estado real por sprint:
  - `S7` marcado en `DONE` con evidencia de `S7_7.1` a `S7_7.5`.
  - `S8` agregado y marcado en `DONE` con resumen breve de objetivo (correlacion logs-traces + OpenAPI + Swagger UI).
  - Hitos de cierre ampliados para incluir explicitamente tracing y OpenAPI/Swagger UI como `DONE`.
- Se actualizo `README.md` en requisitos de Go a `1.24+` para alinearlo con `go.mod`.
- No se realizaron cambios de codigo ni rutas.
